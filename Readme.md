## 15-puzzle game

![Alt text](/ass/ui.jpg?raw=true "UI")

## Controls
Arrow keys to **move**

Space to **shuffle**

S-key to **solve** by computer

## Compile and run
`make` to compile

`./_npuzzle` to run the program

## Warning

The program depends on Raylib for graphics
